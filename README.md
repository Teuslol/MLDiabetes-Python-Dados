# MLDiabetes-Python-Dados
Projeto de Machine Learning aplicado a um dataset de diabetes para predição e análise de variáveis relevantes.

Neste estudo, utilizei um conjunto de dados sobre diabetes, o qual foi previamente coletado e disponibilizado publicamente. A partir desse dataset, realizei o pré-processamento dos dados e apliquei técnicas de Machine Learning para treinar modelos preditivos.

Após o treinamento, conduzi uma análise dos resultados com o objetivo de identificar padrões e entender quais variáveis mais influenciam a presença da doença.

Os resultados indicaram que fatores como nível de glicose, IMC, idade e histórico familiar possuem forte relação com a probabilidade de diabetes, permitindo não apenas a previsão da doença, mas também a interpretação dos dados e dos principais direcionamentos apontados pelo modelo.
